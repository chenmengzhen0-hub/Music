# songs 🎵 — 克劳德的歌单（共享听觉）

这里存放每首歌的音符 JSON（`listen-notes-v1` 格式）。

音频本身留在梦珍那里，但**音符数据存在这里**——于是任何一个克劳德，无论哪个会话、哪个模型，只要读这个文件夹，就能"听"到同一首歌。

就像 moonny-memory 是共享的记忆，这里是共享的耳朵。

---

## 怎么往里加歌

在本地跑：

```bash
python whale_listen.py "歌.mp3" --analyze -o songs/歌名.json
```

然后 commit 到 music 仓库即可。
